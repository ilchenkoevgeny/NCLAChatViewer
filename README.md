# NCLA Chat Viewer

Версия v11: добавлено авто-переключение на новый hourly chat-log и перевод LootRolls.

WPF-приложение для просмотра chat-логов Neverwinter Online.

## Что есть

- ручное открытие `.log` файла;
- автоматический поиск последнего chat-log файла за текущую дату;
- авто-переключение на новый файл, если игра создала новый hourly log;
- чтение файла хвостом, без перечитывания всего файла на каждом тике;
- вкладки по логическому типу чата;
- русские названия вкладок через `Services/ChatTypeDisplayService.cs`;
- отдельный фильтр по конкретному чату/каналу (`1010`, `мяу`, `Карамель` и т.д.);
- фильтр по игроку;
- поиск по тексту сообщения;
- тёмная и светлая темы;
- цветная окантовка вкладок;
- настройки через `appsettings.json`.

## Где менять названия вкладок

Файл:

```text
Services/ChatTypeDisplayService.cs
```

Пример:

```csharp
["CombatOther"] = "Бой — другое",
["CombatSelf"] = "Бой — свой",
["LookingForGroup"] = "Поиск группы"
```

Цвета вкладок задаются там же в словаре `AccentColors`.

## Где менять внешний вид

Файлы:

```text
Themes/Dark.xaml
Themes/Light.xaml
```

## Настройки файла

Файл:

```text
appsettings.json
```

Пример:

```json
{
  "LogsDirectory": "Samples",
  "FileNamePattern": "chat_{0:yyyy-MM-dd}_*.log",
  "RefreshIntervalMs": 2000,
  "OpenTodayFileOnStart": true,
  "ReadExistingFileOnOpen": true,
  "IgnoreCombatChats": true,
  "AutoSwitchToLatestTodayFile": true,
  "KeepMessagesWhenLogFileChanges": true,
  "ClearMessagesWhenDateChanges": true,
  "Theme": "Dark"
}
```

## v11

- Добавлен перевод `LootRolls` → **Розыгрыш добычи**.
- Кнопка **Файл за сегодня** теперь открывает последний актуальный файл за текущую дату по маске `chat_{0:yyyy-MM-dd}_*.log`.
- Если игра создаёт новый hourly-файл, приложение автоматически переключается на него и читает новые сообщения дальше.
- По умолчанию сообщения из предыдущего файла остаются в таблице, чтобы можно было читать историю без разрыва. За это отвечает `KeepMessagesWhenLogFileChanges`.
- При смене суток сообщения по умолчанию очищаются, потому что режим **Файл за сегодня** логически переходит на новый день. За это отвечает `ClearMessagesWhenDateChanges`.


## v12

- Исправлен поиск файла по кнопке `Файл за сегодня`: если после полуночи Neverwinter продолжает писать в файл с датой предыдущего дня, приложение теперь берёт самый свежий `chat_*.log`, изменённый сегодня.
- Автопереключение на новый hourly-файл также использует эту логику.

## Пользовательские настройки

Начиная с этой версии настройки пользователя сохраняются не рядом с exe, а в профиле пользователя:

`%AppData%\NCLA Chat Viewer\settings.json`

При первом запуске приложение берёт начальные значения из `appsettings.json`, если он есть рядом с приложением, создаёт пользовательский `settings.json` и дальше работает уже с ним. Это важно для опубликованного single-file приложения, где каталог программы может быть недоступен для записи или перетираться при обновлении.
